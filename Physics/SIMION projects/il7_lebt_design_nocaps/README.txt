Short man of the process to use the scripts.

1.- Open "gen_pit30_einzel.py"
 1.1.- Customize the for loops to generate a set of PAs.

2.- Edit the batch_config.cvs file.

3.- Execute the simulations.
  3.1.- Double click on solenoid_fe.cmd
    
4.- Results
 4.1.- The ions' end positions and velocities will be stored in "recordings" folder
 4.2.- The stats of the particles will be stored in "stats/stats_<current_date_time>.csv"
       velocities rms, emittances and beam loss
 4.3.- The solenoid_fe_log.txt file contains the log of the simulations.
